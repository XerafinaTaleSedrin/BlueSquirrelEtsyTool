# Selesnya Heliod Vanguard Combo (Historic Bo1)

**Format:** Historic | **Cards:** 60 | **Colors:** GW
**Archetype:** Lifegain Combo with Vanguard Kill
**Status:** Built from confirmed collection (untapped.gg export 2026-03-14)

## Strategy

Infinite combo deck with the strongest kill condition available: Thunderbond Vanguard turns every squirrel token into an X/X copy where X = your creature count. Three pieces on board = infinite 250/250s.

**Combo line:** Soul Warden/Lunarch Veteran + Scurry Oak + Heliod, Sun-Crowned = infinite tokens. With Thunderbond Vanguard in play, each token enters as a copy of Vanguard (P/T = creature count). 250 copies of a 250/250. Attack for 62,500+.

**Conclave Mentor acceleration:** Even without Heliod, Conclave Mentor doubles every counter on Scurry Oak, generating massive boards quickly. Not infinite, but often lethal.

**Fair plan:** Trelasarra scries every lifegain trigger while growing huge. Voice of the Blessed gains flying + indestructible at 10 counters.

## Why This Over the Previous Vanguard Build

The existing squirrel-vanguard-60 runs many 1-ofs (Griffin Aerie, Worldweave, Felidar Retreat, Blessed Sanctuary, Anointed Procession, Parallel Lives, Gaea's Blessing) that are individually powerful but make the deck inconsistent. This version:

1. **Cuts all cute 1-ofs** — no 4-5 CMC cards that sit in hand
2. **Better manabase** — actual dual lands (Temple Garden, Razorverge Thicket) instead of Blossoming Sands + basics only
3. **4x Conclave Mentor** — the secret MVP. Doubles counters so even without Heliod, Scurry Oak generates a massive board. Also distributes its counters when it dies
4. **More consistent** — higher counts of key cards, lower curve, Commune digs for everything
5. **Keeps the Vanguard kill** — still wins with infinite 250/250 Vanguard copies when the combo fires

## Gameplay Notes

**Mulligan:** Keep hands with a lifegain creature (Soul Warden/Lunarch Veteran) + land + either a payoff or Commune. Combo pieces in the opener are a bonus.

**Sequencing:**
- T1: Soul Warden or Lunarch Veteran (lifegain engine first). Commune if no 1-drop creature.
- T2: Conclave Mentor (doubles counters) or Trelasarra (scry + grow)
- T3: Scurry Oak (combo piece) or Thunderbond Vanguard (finisher)
- T4+: Heliod to go infinite, or just attack with massive board

**The kill turn:**
```
Board: Soul Warden + Scurry Oak + Heliod + Thunderbond Vanguard

1. Any creature enters or any lifegain trigger
2. Heliod puts +1/+1 counter on Scurry Oak
3. Scurry Oak creates a Squirrel token
4. Token enters as a COPY of Thunderbond Vanguard
5. Soul Warden gains 1 life (creature entered)
6. Repeat 250 times (Arena token cap)

Result: 250 tokens, each is a Thunderbond Vanguard copy
Each copy's P/T = 254+ (creature count)
Attack for 63,500+ damage.
```

**Without Heliod (Conclave Mentor plan):**
```
Board: Soul Warden + Conclave Mentor + Scurry Oak + Thunderbond

Cast any creature → Soul Warden gains life
If Scurry Oak has any +1/+1 counter source, Mentor doubles it
2 counters = 2 squirrels = 2 Soul Warden triggers
Not truly infinite, but generates 10-20+ Vanguard copies quickly
Each is a 15/15+ — that's lethal.
```
