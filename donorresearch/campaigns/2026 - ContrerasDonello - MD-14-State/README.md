# Campaign Setup Guide

This folder is a template for setting up a new campaign client.

## To Create a New Campaign

1. Copy this entire `_CAMPAIGN_TEMPLATE` folder
2. Rename it using the format: `YYYY - CandidateName - Location`
   - Example: `2026 - Smith - MD-14`
   - Example: `2026 - Johnson - VA-07`
3. Update the `campaign-profile.md` file with campaign details
4. Customize the knowledge-base files for this specific campaign
5. Begin prospect and endorsement research

## Folder Contents

```
YYYY - CandidateName - Location/
├── campaign-profile.md          # Campaign details and candidate info
├── prospects/                   # Donor research for this campaign
│   ├── _PROSPECT_TRACKER.md
│   ├── major-donors/
│   ├── mid-level/
│   ├── bundlers/
│   └── small-dollar/
├── endorsements/                # Endorsement research for this campaign
│   ├── _ENDORSEMENT_TRACKER.md
│   ├── unions/
│   ├── advocacy-organizations/
│   ├── civic-groups/
│   ├── elected-officials/
│   └── media-newspapers/
├── network-maps/                # Relationship mapping
│   └── _NETWORK_OVERVIEW.md
├── compliance/                  # Campaign-specific compliance notes
│   └── campaign-compliance-notes.md
└── knowledge-base/              # Campaign-specific knowledge
    ├── district-profile.md
    ├── candidate-positions.md
    ├── prior-campaign-donors.md
    └── endorsement-patterns.md
```

## After Setup

1. Read the main `CLAUDE.md` in the root folder for workflow instructions
2. Reference the `frameworks/` folder for scoring and evaluation frameworks
3. Use `templates/` for creating new research files
4. Track progress in the campaign's tracker files
